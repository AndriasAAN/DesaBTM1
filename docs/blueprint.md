# **App Name**: Batumarta Satu Digital

## Core Features:

- Berita Desa: Menampilkan berita terbaru dan informasi penting terkait Desa Batumarta 1.
- Profil Desa: Informasi lengkap mengenai sejarah, demografi, geografis, dan potensi Desa Batumarta 1.
- Galeri Kegiatan: Menampilkan foto dan video kegiatan-kegiatan yang dilaksanakan di Desa Batumarta 1.
- Layanan Publik: Informasi dan formulir untuk layanan publik seperti surat pengantar, perizinan, dan lain-lain.
- Direktori UMKM: Mendaftar dan mencari informasi usaha mikro, kecil, dan menengah (UMKM) di Desa Batumarta 1.
- Kontak Admin: Menampilkan informasi kontak admin website (Andrias AAN Setiawan, +628974597885, rahmawan3114@gmail.com).
- Event Calendar: Display the upcoming event information to website users.

## Style Guidelines:

- Primary color: Earthy Green (#6B8E23) to reflect the natural beauty and agricultural landscape.
- Background color: Soft Beige (#F5F5DC) for a clean and modern feel.
- Accent color: Warm Gold (#D4A27A) to highlight important information and calls to action.
- Body font: 'PT Sans', a humanist sans-serif font for readability and a modern feel.
- Headline font: 'Playfair', a modern serif for elegant headers.
- Use modern and clean icons to represent different sections and services.
- Modern and responsive layout that adapts to different screen sizes.
- Subtle transitions and animations to enhance user experience.
